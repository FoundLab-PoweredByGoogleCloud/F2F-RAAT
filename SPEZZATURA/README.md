# Spezzatura T²: Deterministic Reputation Engine

This directory contains the detailed specifications for the **Spezzatura T² engine**.

Within the FoundLab **Auditable Trust Infrastructure (ATI)**, Spezzatura serves as the primary **deterministic reputation calculation engine**. It is a core component of the F2F-RAAT decision framework, which operates under the governance of the main ATI.

Its primary role is to convert objective, PII-free **Facts** into a verifiable reputational score. For the formal specification of how its input vectors are calculated, refer to the documents within this directory.
